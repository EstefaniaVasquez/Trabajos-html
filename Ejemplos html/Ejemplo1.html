Etiquetas HTML

Este ejemplo mustra como combinar algunas de las etiquetas más básicas de HTML5.
Recuerda que es importante entender la diferencia entre ellas.

Etiqueta ul+i

Si listamos personas nominadas al Oscars, dado que el orden no altera el significado, debemos usar ul.

  • Ozlem Vásquez
  • Luhana Oncevay
  • Estefania Vásquez
  • Ángeles Vásquez

Etiqueta ol+i

En el caso de que estemos listando elementos donde el orden es importante, como por ejemplo la tabla de posiciones de la COPA AMÉRICA 2019, debemos usar ol.

  1. Brasil
  2. Perú
  3. Argentina
  4. Chile
